# Rate My Stuff

A full-stack web application for creating and managing reviews. Built with React, TypeScript, Node.js, Express, and MongoDB, deployed on Render.

## Features

- **Create Reviews** - Users can create new reviews with custom content
- **View Reviews** - Browse and read all submitted reviews
- **Edit Reviews** - Modify existing reviews
- **Delete Reviews** - Remove reviews from the application
- **Seamless Data Management** - Real-time data persistence with MongoDB

## Tech Stack

- **Frontend:** React.js, TypeScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Deployment:** Render
